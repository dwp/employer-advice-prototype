Deciding on changes at work (reasonable adjustments)

Changes to how {{ data['employeeName']|default('your employee', true) }} works could be needed. Things like reduced hours, different shifts or avoiding some duties can help.

Changes that might help physical conditions include:

  • wheelchair access
  • a different desk or chair
  • specialist equipment, like a keyboard for people with arthritis

You should ask {{ data['employeeName']|default('your employee', true) }} if any of these changes would help them. You can make suggestions, but it’s important to let them take the lead if possible.

Make the changes if you can

Changes to the workplace are called reasonable adjustments. This means you must make the changes if they are reasonable for your business.

Changes might be reasonable if they are practical, affordable for your business and they won’t harm anyone else.

Examples of workplace adjustments - Equality and Human Rights Commision

If you do not think the changes are reasonable

If the changes are too expensive, there might be help available. Employees can use the Access to Work scheme to pay for some things, but only if you cannot provide them.

Access to Work - support for your employees

If changes are not practical for your business or the type of job, you should try to agree another option together.

Acas - if an employer does not make reasonable adjustments
